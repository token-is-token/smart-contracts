# LLM Share Network Smart Contracts

LLM Share Network 的智能合约部分，负责代币经济、资源分配和治理。

## 技术栈

- **Solidity**: 智能合约编程语言
- **Hardhat**: 开发、测试和部署环境
- **Foundry**: 高性能测试框架
- **OpenZeppelin**: 标准合约库
- **0G Chain**: 目标部署网络

## 安装

```bash
npm install
```

## 编译

使用 Hardhat 编译：

```bash
npx hardhat compile
```

使用 Foundry 编译：

```bash
forge build
```

## 测试

运行 Hardhat 测试：

```bash
npx hardhat test
```

运行 Foundry 测试：

```bash
forge test
```
